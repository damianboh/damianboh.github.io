Ray Diagram of Thin Convex Lens

Enter 2 distances (f,u,v) and 1 height (h1,h2) to calculate the 3rd distance and 2nd height. Distances are measured from the optical centre of lens. To standardize enter only positive values for f,u and h1 so object is always real and upright. If you enter v and h2 together, make sure both have the same sign, either positive or negative. If both height fields are left blank then object height will be set to a default value of 10cm.
